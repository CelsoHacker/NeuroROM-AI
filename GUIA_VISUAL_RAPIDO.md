## 💡 DICA PRO: Como Identificar?

### É ROM de Console se:
- ✅ Arquivo único (.smc, .nes, .gba)
- ✅ Tamanho: geralmente < 32 MB
- ✅ Roda em emulador (RetroArch, etc.)

### É Jogo de PC se:
- ✅ Arquivo .exe ou pasta com vários arquivos
- ✅ Tem arquivos .wad, .pak, .dat, .dll
- ✅ Roda nativamente no Windows/Linux

## ❓ FAQ VISUAL

### "Por que jogos de PC são mais complexos?"

```
ROM DE CONSOLE (1 arquivo):
┌──────────────────────┐
│  jogo_console.smc    │  ← Tudo aqui dentro
│  ┌────────────────┐  │     - Código
│  │ Código         │  │     - Gráficos
│  │ Textos         │  │     - Música
│  │ Gráficos       │  │     - Textos ✅
│  │ Música         │  │
│  └────────────────┘  │
└──────────────────────┘

Fácil de modificar: Abrir, editar, salvar ✅


JOGO DE PC (muitos arquivos):
┌──────────────────────────────────────┐
│  Pasta do Jogo/                      │
│  ├─ jogo.exe          ← Executável   │
│  ├─ dados.wad         ← Dados        │
│  ├─ textures/         ← Gráficos     │
│  ├─ sounds/           ← Áudio        │
│  ├─ language.txt      ← Textos aqui! │
│  ├─ config.ini        ← Config       │
│  └─ mods/             ← Mods         │
└──────────────────────────────────────┘

Complexo: Cada jogo tem estrutura diferente ⚠️
```
